## ❔ How to execute - Como executar

```bash
# Clone repository
$ git clone https://github.com/sunderhus/bluecoding

# After clone, open on your favorite IDE, in my case Visual Studio Code.


$ cd FrontendChallenge && code .

# install all dependencies 
$ yarn

# run project 
$ yarn start

```
